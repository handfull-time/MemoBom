본 제품/서비스는 오픈소스 소프트웨어를 포함하고 있습니다. 각 오픈소스의 저작권은 해당 권리자에게 있으며, 라이선스 조건(고지 유지, 라이선스 전문 제공, 소스 공개 의무 등)은 각 라이선스에 따릅니다.

아래 목록은 pom.xml에 명시된 직접 의존성(direct dependency)을 기준으로 작성되었습니다. Spring Boot Starter 등은 전이 의존성을 포함하므로, 실제 배포물에는 추가 오픈소스가 포함될 수 있습니다.
